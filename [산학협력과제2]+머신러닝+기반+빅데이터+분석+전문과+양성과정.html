산학협엽과제 데이터 설명서 ¶

과제명 : 전기차 고전압 배터리의 주행상태 데이터를 활용한 충전용량 예측¶

차량 시계열 센서 신호를 활용하여 고전압 배터리 SOC 상태에 따른 주행가능거리(충전용량) 예측 모형 생성¶

해야할 일들 ¶

1. 충전 용량 예측(주행거리 값 예측)¶

  • 시간(날짜) 별로 발생되는 SOC 상태 값에 대한 주행거리 데이터 생성
  • 날짜/주행거리/SOC 상태값의 변화량을 활용한 특정 주행거리 시점별 충전용량 예측

2. 충전용량(주행거리)와 연관 있는 주행 패턴 추출¶

  • 차속, YawRate, 종/횡가속도 를 활용한 주행 패턴 추출

분석 요건 설명¶

  1. 데이터 준비 및 사전 프로파일링
    • 시계열 데이터 셋 현황 파악 : 각 채널별 변수 통계값 추출
    • 데이터 설명서
      • 데이터는 ";" 딜리미터로 되어있음
      • 시계열 데이터의 형태로 존재함

  2. 데이터 가공

    • 시계열 데이터 가공 : SOC상태, 주행가능거리, 시간별 값을 활용한 데이터 가공

  3. 분석 모형 생성

    • 시간의 텀으로 발생하는 동일한 SOC 충전용량일 때의 주행가능 거리를 활용한 주행가능 거리 예측
    • 시계열 예측 모형 활용
    • 배터리 성능 열화 인자 발굴
      • 모터토크, 모터RPM, 차속, 브레이크 작동횟수 등

  4. 결과 평가

    • 주행거리(ODO)가 늘어남으로 인해 1만,5만,10만 KM 단위에서의 동일 SOC 상태의 누적주행 가능 거리 예측 모형 생성
